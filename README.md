# Superstore Sales Dashboard (Power BI)

## Project Overview
This project is a Sales Dashboard created using Power BI to analyze sales performance and forecast future trends.

## Features
- Total Sales, Profit, Quantity KPIs
- Region-wise Sales Analysis
- Segment-wise Analysis
- Category & Sub-category Insights
- Shipping Mode Analysis
- State-wise Sales Map
- 15-Day Sales Forecasting

## Tools Used
- Power BI
- Excel Dataset

## Key Insights
- Total Sales: 1.6M
- Profit: 175K
- West region performs best
- Consumer segment dominates
- Phones are top selling product
- Sales forecast remains stable
